# 渲染层级与动画说明

| zIndex | 内容 |
|---:|---|
| 0 | 房间背景 |
| 10 | 黑板等后景道具 |
| 25 | 水晶球、购物袋 |
| 30 | 人物 |
| 45–47 | 搜索环、轨道、旋涡 |
| 60 | 气泡和精灵资料 |
| 61–62 | 商品卡片装饰和飞行拖尾 |
| 70 | 奖励星星和页面主 UI |
| 80 | 顶部与底部可点击组件 |
| 90 | Sheet、Dialog 和错误提示 |

## SEARCHING

扫描环顺时针旋转，轨道环逆时针旋转，整体做轻微呼吸缩放。特效中心应靠近
人物蓝色购物包，而不是屏幕中心。

## PRESENTING

商品内容使用 Compose 卡片。发光框只能作为外围装饰。卡片从购物包位置移动到
页面右侧或下半部。

## ADD_TO_CART

卡片中心从当前位置飞向运行时测得的购物车图标中心，拖尾跟随路径，不能使用
固定终点坐标。
